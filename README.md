HamburgueriaZ 🍔

Projeto desenvolvido no Android Studio como atividade da faculdade.

O aplicativo simula um sistema simples de pedidos para uma hamburgueria, permitindo escolher adicionais, definir quantidade e gerar um resumo do pedido via e-mail.

📱 Funcionalidades
Adicionar ou remover quantidade de hambúrgueres
Selecionar adicionais:
Bacon
Queijo
Onion Rings
Cálculo automático do preço
Geração de resumo do pedido
Envio do pedido por e-mail usando Intent

🛠️ Tecnologias utilizadas
Java
Android Studio
Android SDK
XML (layouts)

Clone o repositório:
git clone https://github.com/SEU-USUARIO/HamburgueriaZ.git
Abra o projeto no Android Studio
Aguarde o Gradle sincronizar
Execute em um emulador Android ou dispositivo físico
